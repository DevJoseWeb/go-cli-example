go-cli-example
==============
Learning Go - This example consumes a github api and returns a user's data.

### Build & Run
```
$ go build
$ ./go-cli-example
Usage: ./go-cli-example [options]
Options:
  -user string
    	Search Users (default " ")

$ ./go-cli-example -user infoslack
```
